Today's jumble

Classic scrambled word game with a perfect amount of brain-tease.

Instructions:

Unscramble these Jumbles, placing one letter on each square, to form ordinary words.
Use TAB key to input the next letter. Use DEL key to delete current input.
The letter will be colored in Tomato when it is wrong, or in SlateBlue when correct.
